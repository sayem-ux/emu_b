Microcode Mnemonics:

MAR
An 8-bit register created using flip flops that contains the address that RAM will be read/written from.
PC
An 8-bit counter of the CPU that keeps track of the current instruction.

Fetch Stages:

  1. FETCH1: MAR<=PC
  2. FETCH2: MDR<=M[MAR], PC<=PC+1
  3. FETCH3: IR<=MDR

Instructions:

  • NOP
  • LOAD
  • LOADI
  • STORE
